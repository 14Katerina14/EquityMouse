# EquityMouse

Platform for analyzing and visualizing public data on institutional holdings, insider transactions, and company financial metrics.

## Project structure

- `frontend` - React Native app
- `backend` - Node.js + Express API
- `db` - future PostgreSQL setup

## Current app scope

- mock login that accepts any credentials
- home screen with searchable market cards
- detailed asset views with live and fallback market data flows
- guidebook / educational screen
- backend scraping for quotes and selected metrics

## Notes

- scraping source mappings are kept in ignored local config files and are not intended for public version control
- environment keys are excluded from git
